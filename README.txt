ABOUT
─────
A modern, fully-responsive static portfolio website for the
DesignsBySSM creative design studio. Built with HTML, CSS,
and CSS animations only — no JavaScript required.


PROJECT STRUCTURE
─────────────────
DesignsBySSM-Website/
│
├── index.html          ← Main website file (open this)
│
├── css/
│   └── style.css       ← All styles, animations & responsive
│
├── images/
│   └── (placeholder)   ← Add your portfolio images here
│
└── README.txt          ← This file


HOW TO RUN
──────────
1. Download / unzip the folder
2. Open in VS Code (or any text editor)
3. Double-click index.html — it opens directly in your browser
4. No build tools, no npm install, no server required


SECTIONS INCLUDED
─────────────────
• Navigation       — Fixed glassmorphism navbar
• Hero             — Animated gradient background + CTA
• Ticker           — Scrolling services marquee
• About            — Studio intro + visual card
• Services         — 8 animated hover cards
• Portfolio        — 6-item grid with hover overlays
• Why Choose Us    — 6 reasons + stats row
• Contact          — Info items + visual panel
• Footer           — Clean minimal footer


DESIGN SYSTEM
─────────────
Fonts    : Bebas Neue (display), Syne (headings), DM Sans (body)
           → Loaded from Google Fonts (internet required)
Colours  : Deep black (#050508) base
           Electric cyan  (#00f5d4) primary accent
           Neon magenta   (#f700ff) secondary accent
           Gold           (#ffd60a) tertiary accent
Effects  : Glassmorphism cards, CSS mesh animations,
           gradient text, animated borders, ticker marquee

CUSTOMISATION
─────────────
1. Replace contact details in index.html (Instagram / Email / WhatsApp)
2. Swap placeholder text with your real project names & descriptions
3. Add real portfolio images: place in /images/ and update src=""
4. Update stats (500+ projects, 200+ clients etc.) to match reality
5. Change accent colours in css/style.css under :root variables
